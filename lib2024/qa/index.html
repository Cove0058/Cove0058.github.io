  10. Qus What is the use of name and value attributes? Give two examples

    Ans: The name and value attributes are used to make forms in HTML. The name attribute is used as the reference name for the data completed and submitted. The value attribute is default data that will be submitted unless the user inputs alternative data.
    Example 1: <input type="text" id="fname" name="fname" value="John"> In this example, the input name is called "fname" and the default data that will be submitted is going to be "John". The user can, however, replace this value with any of other name if required.
    Example 2: <input type="text" id="lname" name="lname" value="Doe"> In this example, the input name is called "lname" and the default data that will be submitted is going to be "Doe". The user can, however, replace this value with any of other name if required.

  14. Qus Create CSS keyframes block called "spin" that rotates an element.

    Ans: The code to be used is as follows:

    <style>
    @keyframes spin {
    from {
    transform: rotate(0deg);
    }

    to {
    transform: rotate(360deg);
    }
    }
    </style>

  15. Qus Create CSS animation block to rotate an element that has the following properties :
